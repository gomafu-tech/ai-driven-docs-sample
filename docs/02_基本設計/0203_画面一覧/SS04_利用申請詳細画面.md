# SS04: 利用申請詳細画面

**アクセス方法**: 利用申請一覧 → 申請ID クリック  
**関連業務フロー**: [BF003: 利用申請承認・否認](02_基本設計/0201_業務フロー/BF003_利用申請承認・否認.md)

## 主要機能
1. **申請詳細表示**: 利用申請の全詳細情報表示
2. **証明書確認**: アップロードされた証明書類の確認
3. **承認・否認**: 申請の承認・否認処理

## 操作機能

### 承認・否認処理
- **承認**: 申請を承認して会員登録
- **否認**: 申請を否認（否認理由入力）
- **保留**: 追加確認が必要な場合の保留

### その他操作
- **一覧に戻る**: 申請一覧画面に戻る
- **申請者連絡**: メール送信画面に遷移
