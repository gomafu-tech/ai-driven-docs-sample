# SW02: 団体会員利用申請画面

**アクセス方法**: Webサイトのトップページから「新規利用申請」→「団体会員申請」  
**関連業務フロー**: [BF002: 団体会員利用申請](02_基本設計/0201_業務フロー/BF002_団体会員利用申請.md)

## 主要機能
1. **団体情報入力**: 団体名、代表者情報、活動内容等の入力
2. **証明書類アップロード**: 団体設立証明書、活動証明書等の登録
3. **入力内容確認**: 入力情報の確認・修正
4. **申請提出**: 利用申請の正式提出
