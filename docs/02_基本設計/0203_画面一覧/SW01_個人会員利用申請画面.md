# SW01: 個人会員利用申請画面

**アクセス方法**: Webサイトのトップページから「新規利用申請」→「個人会員申請」
**関連業務フロー**: [BF001: 個人会員利用申請](02_基本設計/0201_業務フロー/BF001_個人会員利用申請.md)

## 主要機能
1. **個人情報入力**: 氏名、生年月日、連絡先等の基本情報入力
2. **身分証明書アップロード**: 運転免許証、マイナンバーカード等の画像ファイル登録
3. **入力内容確認**: 入力情報の確認・修正
4. **申請提出**: 利用申請の正式提出
